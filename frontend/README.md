# 啄木鸟健康管理助手 APP

一款全方位的健康管理应用，提供医疗服务、健康监测、健康商城和个人中心四大核心功能模块。

---

## 🏥 服务模块 (Service)
提供专业医疗健康服务入口，包含以下功能：

### 1. 医学百科
- **文章分类浏览**：按科室/疾病分类的健康知识库
- **专业内容**：由医疗机构认证的医学文章
- **搜索功能**：支持关键词搜索医学内容
- **收藏功能**：用户可收藏重要文章

### 2. 健康评测
- **风险评估**：提供慢性病风险评估问卷
- **测评报告**：生成可视化健康评分报告
- **历史记录**：保存历次测评结果对比
- **建议方案**：基于评测结果的健康建议

### 3. 我的权益
- **会员服务**：显示当前会员等级及权益
- **服务记录**：查询历史服务使用情况
- **权益兑换**：积分兑换健康服务
- **到期提醒**：会员服务到期提醒

### 4. 疾病管理
- **儿科健康**：儿童成长里程碑/疫苗接种提醒
- **高血压管理**：血压记录/用药提醒
- **糖尿病管理**：血糖监测/饮食建议
- **心理健康**：心理测评/减压方案

---

## ❤️ 健康模块 (Health)
个人健康数据监测中心

### 1. 健康看板
- **数据总览**：血压、血糖、体重、运动等关键指标
- **趋势图表**：支持按日/周/月查看数据变化
- **异常预警**：指标超出范围时主动提醒

### 2. 指标监测
- **血压管理**：
  - 记录收缩压/舒张压
  - 自动计算平均血压
  - 生成血压波动曲线
- **血糖管理**：
  - 空腹/餐后血糖记录
  - 糖化血红蛋白追踪
  - 血糖变化趋势分析
- **体重管理**：
  - BMI自动计算
  - 体脂率记录
  - 减肥/增重目标设定

### 3. 运动健康
- **步数统计**：对接手机健康数据
- **运动记录**：记录跑步/健身等运动
- **消耗计算**：估算卡路里消耗
- **目标管理**：设置每日运动目标

---

## 🛍️ 商城模块 (Store)
健康相关商品购买平台

### 1. 商品分类
- **健康监测**：血压计、血糖仪等
- **康复护理**：理疗仪、护具等
- **保健食品**：维生素、益生菌等
- **运动健康**：健身器材、运动装备

### 2. 商品详情
- **高清展示**：多角度商品图片
- **参数说明**：详细产品规格
- **用户评价**：真实购买评价
- **促销信息**：折扣/满减活动

### 3. 购物服务
- **购物车**：多商品合并结算
- **订单管理**：查看订单状态
- **物流跟踪**：快递信息查询
- **售后服务**：退换货申请

---

## 👤 我的模块 (User)
个人账户中心

### 1. 账户管理
- **个人信息**：修改头像/昵称
- **健康档案**：完善健康信息
- **家庭成员**：添加家人健康档案
- **账号安全**：修改密码/绑定手机

### 2. 健康设置
- **指标预警**：设置各指标预警值
- **单位偏好**：选择血压/mmHg等
- **数据同步**：设置自动同步频率
- **提醒设置**：用药/测量提醒

### 3. 系统设置
- **主题切换**：明亮/暗黑模式
- **通知管理**：各类消息通知设置
- **清理缓存**：释放存储空间
- **关于我们**：应用版本信息

---

## 🔧 技术栈
- 前端框架：Vue 3 + uni-app
- 状态管理：Vuex
- UI组件：uni-ui
- 构建工具：Webpack
- 代码规范：ESLint + Prettier

## 📱 兼容性
- 支持iOS/Android双平台
- 适配各种屏幕尺寸
- 支持微信小程序

## 🚀 开发指南
1. 安装依赖：`npm install`
2. 开发模式：`npm run dev:mp-weixin`
3. 生产构建：`npm run build:mp-weixin`
4. 代码检查：`npm run lint`
